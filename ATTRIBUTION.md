# Art Attribution

## Vettlingr 32x Tileset

The tile art in `src/render/tileset-assets/` is sourced from the
**Vettlingr 32x** tileset for Dwarf Fortress, created by **vettlingr**.

- Original thread: https://www.bay12forums.com/smf/index.php?topic=172078.0
- Download page: https://dffd.bay12games.com/file.php?id=14072
- Mirror used for sourcing: https://github.com/DFgraphics/Vettlingr

The original tileset's README states it is intended for use within
Dwarf Fortress only, and that use outside DF requires permission from
the artist. **The artist has granted explicit permission for this use**
in this non-commercial personal project. This permission is scoped to
this project as discussed directly with the artist; it should not be
assumed to extend to other projects, commercial use, or redistribution
without separately confirming with the artist.

Tiles used here are a small selection, cropped from the original
sheets and in some cases recolored via canvas multiply-tinting (see
`src/render/TileCache.ts`) to represent different in-game materials.
