# Reference Art - 3x3 Workshops

Four reference images provided 2026-06-23, in the same Vettlingr-style
tileset aesthetic already used elsewhere in the game (see
`src/render/tileset-assets/`, `ATTRIBUTION.md`). Intended as visual
reference for the still-queued Smelter/room-state work described in
`docs/OPEN_QUESTIONS.md` (search for "The Smelter").

- `Magma_forge.png` - tools/anvil workshop setup
- `Metalsmith_s_forge.png` - a bellows-driven forge room
- `Smelter.png` - a glowing smelter with ingot molds
- `Wood_furnace.png` - a wood-fired kiln/furnace with barrels

Not yet used anywhere in the actual game - held here until the
Smelter's open design questions (purified material identity,
drop-chance, build cost, what it unlocks) get real answers, at which
point one or more of these likely becomes the basis for that room's
sprite work.
