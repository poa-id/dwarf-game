## True-metals

Never used as fuel or smithing input — only a perk-tree currency, earned via the [Smelter](Smelter-and-True-Metals.md) or [trading](Trade-Hall.md).

| Material | Tier |
|---|---|
| True Copper | 1 |
| True Iron | 2 |
| True Deepstone | 3 *(purification track not yet built)* |

## Wood ladder

| Material | Tier | Source | Feeds into |
|---|---|---|---|
| Cave-Root Wood | 1 | Root Tangle | Copper tools |
| Ironwood | 3* | Garden (Ironwood Sapling, Herblore 8) | Iron tools |
| Gemwood | 4 | Garden (Gemwood Tree, Herblore 15) | Deepstone tools |
| Ancient Seed | 2 | Root tangle drop / seed chest / trade | Plants Ironwood |
| Ancient Heartwood Seed | 3 | Trade only | Plants Gemwood |

*Ironwood's internal tier number (3) predates the wood-ladder framing and doesn't line up cleanly with its actual rung (2nd) — kept as-is to avoid churn.
