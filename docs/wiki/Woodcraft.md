# Woodcraft

Active skill. Mechanically identical to Mining — same strike mechanic, axes instead of pickaxes. Covers both cutting raw wood and (conceptually) processing it into planks; there's no separate "refining" skill for wood.

There's no surface in this game — wood comes from cave-root tangles and underground wood formations, not trees on a hillside.

## Wood nodes

| Node | Material | Level req. | Base XP | Base yield | Base success | Notes |
|---|---|---|---|---|---|---|
| Cave-Root Tangle | Cave-Root Wood | 1 | 7 | 1 | 88% | Infinite — never depletes |

This is the other starter node (alongside the copper vein), sitting in the Hearth Hall. Also infinite, for the same reason: exhausting it used to be a permanent dead end.

## Wood ladder

Beyond plain Cave-Root Wood, better wood is grown, not chopped — see [Herblore & The Garden](Herblore-and-Garden.md) for Ironwood and Gemwood, which are required for higher tool tiers.

| Wood tier | Material | Source | Required for |
|---|---|---|---|
| 1 | Cave-Root Wood | Woodcraft (root tangle) | Copper tools |
| 2 | Ironwood | Garden (Herblore 8) | Iron tools |
| 3 | Gemwood | Garden (Herblore 15) | Deepstone tools |
| 4–6 | Stonewood / Emberwood / Voidwood | *(designed, not built)* | — |

Gems don't drop from wood nodes — only Mining's rock veins carry that mechanic.
