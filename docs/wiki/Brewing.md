# Brewing

> **Not yet playable.** The skill exists in the game's data, but no recipes exist and no XP can currently be earned. This page describes the intended design so you know what's coming.

## The plan

Brewing is meant to close the loop that starts in the Garden: grow shrooms/herbs → carry them to the kiln or a future brewing station → brew ale → drink it for a buff. Higher-tier ales should give bigger buffs. Brewing itself levels from the act of brewing.

## Current status

- Skill tab is hidden until first XP is earned (which currently never happens)
- No brew recipes defined
- No brewing station built

Check [Herblore & The Garden](Herblore-and-Garden.md) for the crop side of this loop, which *is* playable today.
